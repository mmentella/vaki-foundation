# Pilot Validation: SPY Momentum Manifestation

## 1. Record Status

This is a post-closure empirical validation record for the exercised scope of VAKI Foundation.

Canonical verdict:

```text
FOUNDATION VALIDATED BY PILOT
```

This record preserves Foundation-level validation evidence from one completed Manifestation. Validation evidence is not proof. One pilot is not universal validation. This record is not a new canonical semantic owner and not a reopening of M1.

## 2. Validation Subject

The pilot was a bounded research Manifestation investigating a frozen SPY momentum hypothesis using daily adjusted-close observations from `2015-01-01` through `2019-12-31`.

The Foundation validation subject is the pilot's use of Foundation semantics within that bounded inquiry. The scientific outcome of the SPY hypothesis is not the subject of Foundation validation.

The validation verdict applies to the semantics represented by that assessed revision. It does not imply automatic applicability to future Foundation revisions that materially change relevant semantics.

## 4. Validation Basis

The completed pilot preserved a traceable inquiry chain:

```text
Question
-> Hypothesis
-> Experiment
-> Observation
-> Evidence
-> Decision
```

The inquiry artifacts were frozen, review-gated, traceable, and covered by integrity checks. The pilot test suite passed 29 tests.

Artifact hashes remain in the pilot repository where needed for direct traceability. They are not reproduced here because this Foundation record records the validation consequence, not the full pilot audit.

## 5. Validated Foundation Semantics

Within this pilot's exercised scope, the following Foundation semantics were validated:

- Question as orienting uncertainty rather than Requirement;
- Hypothesis as provisional and challengeable;
- Experiment as an arrangement creating conditions for possible Observation;
- Observation as preserved reality-facing record rather than Evidence;
- Data as not automatically Observation or Evidence;
- Evidence as an interpreted relation that changes uncertainty;
- Decision as a bounded commitment rather than Action;
- Foundation/Manifestation boundary;
- domain interpretation;
- domain extension;
- implementation freedom;
- semantic ownership;
- traceability across the inquiry;
- preservation of contextual limits.

This validation evidence is scoped to the concepts and relationships actually exercised by the pilot.
